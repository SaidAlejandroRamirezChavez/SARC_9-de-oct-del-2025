¿que usamos?

Flask

Flask es un microframework de Python que sirve para crear aplicaciones web rápidamente y de forma sencilla. Es minimalista, ligero y flexible, proporcionando las herramientas básicas para el enrutamiento de URLs y el manejo de la representación de páginas, pero permite escalar a aplicaciones complejas mediante extensiones. Se usa principalmente para construir APIs, microservicios y prototipos web

python

Python es un lenguaje de programación de propósito general, gratuito y de código abierto, conocido por su sintaxis fácil de leer y su amplia aplicabilidad, que sirve para crear sitios web, software, automatizar tareas, y realizar análisis de datos, ciencia de datos, inteligencia artificial y machine learning. Su popularidad se debe a su versatilidad, su gran comunidad de desarrolladores que aportan librerías extensas, y su facilidad para principiantes.

HTML

HTML, que significa Lenguaje de Marcado de Hipertexto (HyperText Markup Language), es el lenguaje estándar utilizado para crear y estructurar el contenido de las páginas web, desde textos y títulos hasta imágenes y formularios. Funciona mediante etiquetas y atributos que le indican al navegador cómo mostrar los diferentes elementos de una página web, creando la estructura que el usuario final ve en su navegador.
